# With Class Server

## 1. 개요 및 아이디어 도출
코로나 발생으로 인한 비대면 전환으로 모든 대학 또는 온라인 강의 사이트에서 온라인 강의의 수요가 증가, 이에 강의에 대한 지루함과 학우과의 적은 교류를 해결하고자 With Class를 만들게 되었다.
